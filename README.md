# yukiotechblog

Minimal SvelteKit blog hosted on Cloudflare Pages.

## Prerequisites

- Nix with Flakes enabled
- direnv (optional)

## Setup

```bash
nix develop
```

If you use direnv:

```bash
direnv allow
```

## Development

```bash
just dev
```

Open:

```text
http://127.0.0.1:5173/
```

## Build

```bash
just build
```
